# The HQ Consulting

Modern dark-themed property advisory website for Madinah real estate investments.

## Features

- Dark modern theme with emerald accents
- Plus Jakarta Sans + DM Sans typography
- Framer Motion animations with scroll progress
- Fully responsive design (mobile-first)
- Property cards with images
- Contact form with validation
- Smooth scroll and parallax effects

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- shadcn/ui components

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

Deploy to Vercel:

```bash
npm i -g vercel
vercel deploy
```

Or connect your GitHub repository to Vercel for automatic deployments.
